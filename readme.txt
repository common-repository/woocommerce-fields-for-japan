== Description ==

WooCommerceを日本向けに住所の並びなどを変更します。
名前の名字・名前の並び順、住所を郵便番号・県・市町村・番地の並び順に変更します。

このプラグインは、**WooCommerce**([http://wordpress.org/extend/plugins/woocommerce/](http://wordpress.org/extend/plugins/woocommerce/))上で機能します。

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin.
